
# Stroop Task

This repository contains a basic implementation of a Stroop Task using HTML and JavaScript.

## Features

- 5 practice trials
- 2 blocks of 64 trials each
- Balanced congruent/incongruent trials
- 500ms fixation cross before each stimulus
- Stimuli remain on screen until participant response
- Keyboard input: R (Red), G (Green), B (Blue), Y (Yellow)
- Records participant ID, response time, and accuracy
- Summary by condition (congruent vs. incongruent) displayed at the end
- CSV download of trial data

## How to Use

1. Open `stroop_task.html` in a web browser.
2. Enter your participant ID to begin.
3. Complete the practice trials followed by two blocks of trials.
4. At the end, view your performance summary and download the results.

## File

- `stroop_task.html`: Single HTML file containing all logic and styling.
